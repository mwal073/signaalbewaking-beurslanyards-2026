## Besteloverzicht

### Variant A: Telefoon Crossbody Lanyard — 1.500 sets
Per set (verpakt in transparante OPP-zak):
1. **Lanyardkoord**: 25mm breed polyester, 1500-1600mm verstelbare lengte (crossbody), sublimatie bedrukking (dye-sub, dubbelzijdig), Pantone 200 C rode basis, witte opdruk. Draagkracht ≥5kg.
2. **Metalen karabijn**: Zinklegering, geborsteld zilver of mat zwart PVD, ≥25mm opening, draaikoppeling, geschikt voor smartphone (200-250g). Robuuste uitstraling — GEEN goedkoop plastic.
3. **Veiligheidsbreeksluiting**: Bij de nek, mat zwart, breakkracht ≥3kg
4. **Telefoon patch (TPU platte tab)**: Wit TPU + nylon versterking, 0,5mm dik, tab 50×15mm met 10mm port-insteek die door charging port opening schuift. Metalen D-ring (∅7mm) voor koordbevestiging. Rode logo-opdruk op zichtbare zijde. Trekkracht ≥3kg. Compatibel met iPhone 14-16, Samsung S23-S25, Pixel.
5. **Marketingkaartje**: 90x130mm, 350gsm, dubbelzijdig, CMYK druk, mat laminaat met soft-touch afwerking

### Variant B: Badge/Sleutelhanger Lanyard — 1.000 sets
Per set (verpakt in transparante OPP-zak):
1. **Lanyardkoord**: 25mm breed polyester, ±900mm standaard lengte, sublimatie bedrukking (zelfde ontwerp als Variant A). Draagkracht ≥5kg.
2. **Metalen draaihaak**: Zinklegering, geborsteld zilver of mat zwart, ≥20mm opening, robuuste uitstraling
3. **Veiligheidsbreeksluiting**: Bij de nek, mat zwart, breakkracht ≥3kg
4. **Marketingkaartje**: Zelfde als Variant A

## Specificaties

- **Koordkleur**: Pantone 200 C (HEX #C41E24, CMYK C5 M100 Y95 K2)
- **Drukmethode**: Sublimatie (full dye-sub) op polyester koord
- **Lettertype**: Montserrat ExtraBold (omgezet naar contouren in artwork)
- **Hardware**: MOET zinklegering of roestvast staal zijn — GEEN plastic clips. Robuuste/security uitstraling vereist. Alle metaaldelen ≥2mm dikte. Geborsteld zilver of mat zwart PVD.
- **TPU patch**: Platte tab door charging port opening. Moet passen in iPhone 14-16, Samsung S23-S25, Pixel hoesjes. Trekkracht ≥3kg.
- **Telefoonveilig**: Alle onderdelen moeten veilig een smartphone van 250g dragen bij dagelijks crossbody gebruik.
- **Verpakking**: Elke set in OPP-zak. 4-5 buitendozen, max 20kg per doos.
- **Verzending**: DDP luchtvracht naar Nederland.
